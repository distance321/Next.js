## 项目架构
以Next.js为脚手架,antd作为组件，使用Reactjs进行开发的ssr应用。

``` bash

# 安装依赖
npm install

# 在localhost:3000端口带有热编译的起服务方式
npm run dev

# 用于生产环境的压缩后的版本的build脚本
npm run build

# 生成html文件
npm run export


```
